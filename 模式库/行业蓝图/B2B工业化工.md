## 行业特征

### 业务模式
- 制造商直供（不通过贸易商，跨境到终端工厂主）
- 决策周期长（3-6 个月，技术评估 + 试样 + 报价 + 谈判 + 下单）
- 客单价高（$10K-$1M 单笔）
- 复购周期长（设备 5-10 年，原料类 6-12 个月）

### 用户特征
- B2B 决策者：工厂主 / 采购经理 / 总工程师
- 多人决策（采购+技术+财务三方批准）
- 跨时区、跨语言（英语为商业语言，但本地语言增加信任）
- 重视技术参数 + 真实案例 + 资质

### 核心搜索关键词类型
- **产品规格类**："PVA glue PE-1400 spec" / "EPS block molding machine 1400×1500"
- **应用场景类**："PVA glue for fire door bonding" / "EPS for cold chain insulation"
- **对比购买类**："PVA glue vs epoxy resin" / "block vs continuous EPS pre-expander"
- **故障诊断类**："PVA glue cure time too long" / "EPS bead density inconsistency"
- **行业新趋势类**："2026 EPS recycling regulation" / "PVA latex new formula 2026"

## 行业特有的 5 个坑

1. **不要把博客写成"产品广告"** — B2B 用户讨厌硬广，要写"技术问题解决方案"
2. **必须有真实数据** — 浮夸宣传词（"world-class" / "leading"）会被 B2B 用户秒看穿
3. **重视资质证书图** — ISO/SGS/CE 证书扫描图比 100 句宣传有用
4. **客户案例必含数字** — "我们客户产能提升 30%" 比"我们客户很满意"强 100 倍
5. **多语种必须本地化关键词** — 不能直译英文，每语种独立调研（FR/ES/PT/AR/RU 都不一样）
