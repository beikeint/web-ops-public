## 关键成功要素（可复制方法论）

### ① 多语种 SEO 部署的"完整闭环"，不只是翻译
完整链路缺一不可：
1. **内容翻译**（核心产品页 → 关键词本地化）
2. **schema 本地化**（不只翻译可见文本，schema 的 description/name 也要翻）
3. **hreflang 集合补全**（每加一个语种必须更新所有现有页的 alternates，否则集合断裂）
4. **canonical 对齐**（每语种页 canonical 指向自己）
5. **sitemap 多语种登记**（XML sitemap 加 xhtml:link 元素）
6. **IndexNow 推送**（不只 Google，还有 Bing/Yandex，加速冷启动）
7. **GSC sitemap 重提交**（给 Google 信号"有更新"）

### ② "新站 / 新语种沙盒期"识别 + 等待哲学
- 4-22 部署到 4-25 才大规模生效（**3 天沙盒发酵期**）
- 沙盒期内 **不做大改动**让 Google 稳定收录
- 简报里专门标注 "新站发酵期" / "波动正常"，避免误判异常

### ③ 跨语种延伸效应：核心语种带飞辅助语种
- 4-22 部署 FR/ES/PT，4-25 不只这三种生效
- **阿语和俄语展示同步出现**（虽然 AR/RU 是 4-23 才开始翻译）
- 推测：FR/ES/PT 部署引起 Google 全面重抓，包含其他语种的 sitemap 入口

### ④ MCP 数据交叉验证（避免假数据）
- search-analytics MCP 拿 GSC 原始 query
- 过滤带 `-site:` 假关键词（Google Search Console 数据规范）
- "平均排名"标注，禁用"排名第 1"等夸大表述

## 失败 / 待优化点（诚实记录）

### ① CTR 低（0.8%）— 高排名低 CTR 问题
- 平均排名 10.9 但 CTR 仅 0.8%
- 原因推测：Title/Description 不够吸引点击 / 缺品牌差异化卖点
- **下一步**：用 `/CTR client-D` 跑 P5 引擎，对 top 5 机会页批量重写

### ② 转化为 0（7 天 0 个 WhatsApp/email/phone/quote 点击）
- 流量是有了（123 展示），但转化路径未生效
- 原因推测：CTA 不够清晰 / Trust signal 不足 / 多语种页 CTA 是否本地化
- **下一步**：CRO 介入（cro-suite skill 跑），分析 funnel + heatmap

### ③ 客户配合项缺失
- 缺：4 张证书扫描图 + LinkedIn 公司页 + 3 个真实案例
- 影响：E-E-A-T（Experience/Expertise/Authority/Trust）信号不足，AI Overviews 引用难

### ④ 内容量偏低
- 截至 4-26 仅 8 篇博客
- **下一步**：content-production skill 强制激活，每周 ≥ 2 篇

## 复盘到方法论：跨语种 SEO 7 步 SOP

```
1. 关键词调研（每语种独立）→ 不直接翻译英文关键词
2. 翻译产品页核心内容 + 本地化 schema
3. hreflang 集合全网状互引（含 x-default + 自引用）
4. canonical 必须自指
5. sitemap.xml 加 xhtml:link 元素
6. IndexNow 批量推送（Bing + Yandex 都推）
7. GSC sitemap 重提交 + URL Inspection 重抓
8. 等待 3-7 天沙盒期,期间不做大改动
9. 7 天后看 GSC 数据是否生效,生效则扩展下一语种
```
