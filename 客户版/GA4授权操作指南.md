## 操作步骤（只需2步，约2分钟）

### 第一步：在 Google Cloud 中启用 GA4 API

1. 打开 https://console.cloud.google.com
2. 确认顶部项目名称是之前创建的（如 `example-seo`）
3. 左侧菜单 → **API 和服务** → **库**
4. 搜索 `Google Analytics Data API`
5. 点击 **"Google Analytics Data API"**（注意不是 "Google Analytics Admin API"）
6. 点击蓝色的 **"启用"** 按钮

> **看到"管理"而不是"启用"？** 说明已启用，直接下一步。

### 第二步：在 GA4 中添加服务账号

1. 打开 https://analytics.google.com
2. 左下角 **管理**（齿轮图标）
3. 在"账号"列下，点击 **"账号访问权限管理"**
4. 点击右上角蓝色的 **"+"** 按钮 → 选 **"添加用户"**

填写信息：

| 项目 | 填什么 |
|------|--------|
| 电子邮件地址 | `gsc-reader@example-seo.iam.gserviceaccount.com` |
| 角色 | 选 **"查看者"** |

5. 取消勾选"通过电子邮件通知新用户"（服务账号没有邮箱收不了邮件）
6. 点击 **"添加"**

> **找不到"账号访问权限管理"？** 可能是因为您不是GA4的管理员。请联系您公司管理GA4的同事来操作。

## 常见问题

### Q1：安全吗？
完全安全。"查看者"角色只能**读取**数据，无法修改、删除任何内容。您可以随时在GA4管理中移除。

### Q2：会影响GA4数据统计吗？
不会。API读取是独立于网页浏览的，不会产生任何虚假流量或影响数据准确性。

### Q3：需要额外付费吗？
不需要。GA4 Data API 免费使用，在标准配额内完全够用。

### Q4：以后不想授权了怎么取消？
GA4 → 管理 → 账号访问权限管理 → 找到服务账号 → 点击右侧"移除"。
