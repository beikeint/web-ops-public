# 网站运营-web-ops — 每日工作看板

> 这是你的智能体每天在做的事。透明可见，随时可查。

---

## 每日工作流（两步走）

```
08:00  第一步：必做7项巡检（约15分钟）
       ┌─ ☐ 网站可用性    → 网站能打开吗？速度正常吗？
       ├─ ☐ GSC异常       → Google发现问题了吗？
       ├─ ☐ 排名波动      → 关键词排名掉了吗？
       ├─ ☐ 竞品扫描      → 竞品今天做了什么？
       ├─ ☐ 昨日数据      → 流量和询盘正常吗？
       ├─ ☐ 索引提交      → 新页面通知Google了吗？
       └─ ☐ 问题修复      → 发现问题，当场修好
       
       ↓ 修完或无问题

09:00  第二步：主动增长（约30分钟）
       每天做1-2件让你的网站变好的事：
       
       周一 → 优化1个页面的搜索排名表现
       周二 → 给1篇文章补充更深度的内容
       周三 → 补1-2条内部链接
       周四 → 让1个页面对AI搜索更友好
       周五 → 优化1个转化环节（按钮/表单/路径）
       周六 → 抢占Google特殊展示位 + 回应竞品动作
```

---

## 你能看到的产出

### 每天
- 巡检日报（1页，有问题标红，修了的标绿）
- 增长任务记录（做了什么+为什么+效果）

### 每周
- 周检报告（100分健康评分 + 已修复问题清单）
- 关键词排名变化表
- 竞品动态摘要

### 每月
- 月度运营报告（数据分析+工作总结+下月计划）
- 本月新博客2篇
- 技术优化记录

### 每季度
- 季度深度复盘（ROI+趋势+策略调整）
