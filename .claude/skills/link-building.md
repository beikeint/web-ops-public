# 外链建设技能

> 触发指令：`[客户名] 外链策略` / `[客户名] 外链检查`  
> 执行频率：每月1次策略评估，持续执行  
> 人工确认：⏸️ 外链执行计划需确认

---

## 为什么需要外链

Google排名三要素：内容质量 + 技术SEO + **外链权威性**。

没有外链，排名天花板受限。尤其对B2B外贸站，来自行业网站的高质量外链是突破竞品排名的关键杠杆。

**注意**：只做白帽外链，不买链接、不做PBN、不做垃圾目录提交。

## MCP 调用链

### 外链现状分析（`[客户名] 外链检查`）

```
Step 1: 加载客户
        → client-manager.get_client(client_id)

Step 2: 检查现有外链
        → fetch("https://www.google.com/search?q=link:域名") — 粗略检查
        → 或通过GSC"链接"报告数据（如有CSV）
        → 统计：外链总数、来源域名数、锚文本分布

Step 3: 竞品外链对比
        → 分析竞品被哪些网站链接
        → 识别：竞品有但我们没有的外链来源

Step 4: 输出外链分析报告
```

### 外链策略规划（`[客户名] 外链策略`）

```
Step 1: 基于客户行业，识别外链机会

Step 2: 按渠道分类规划

    2a. 行业目录（最快见效）
        → 识别客户行业的在线目录
        → 如：ThomasNet, Alibaba, Made-in-China, GlobalSources, Kompass
        → 检查客户是否已注册 → 未注册的列入提交计划

    2b. Guest Post 机会
        → fetch(行业博客/媒体) → 找到接受投稿的网站
        → 如：EPS行业媒体、包装行业博客、制造业论坛
        → 输出：网站名+提交方式+建议主题

    2c. 合作伙伴链接
        → 客户的供应商/客户/认证机构是否有网站
        → 是否可以互换链接或在合作伙伴页面获得链接
        → 如：原材料供应商网站、行业协会会员页

    2d. 内容引用链接
        → 撰写高质量行业数据/研究报告
        → 让其他网站自然引用（需要内容真正有价值）

    2e. HARO / Journalist Outreach
        → 订阅 HARO（Help a Reporter Out）
        → 以行业专家身份回答记者问题 → 获得媒体链接

Step 3: 输出执行计划（⏸️ 需确认）
```

## 外链质量标准

### 要做的（白帽）

| 类型 | 权威性 | 难度 | 周期 | 效果 |
|------|--------|------|------|------|
| 行业目录 | 中 | 低 | 1-2周 | 基础权威 |
| 行业媒体投稿 | 高 | 中 | 2-4周 | 高质量外链 |
| 合作伙伴 | 中-高 | 低 | 1周 | 相关性强 |
| 行业协会 | 高 | 中 | 2-4周 | 信任信号 |
| 数据报告被引用 | 最高 | 高 | 1-3月 | 最有价值 |

### 绝对不做的（黑帽/灰帽）

- ❌ 购买链接
- ❌ PBN（私有博客网络）
- ❌ 垃圾评论链接
- ❌ 低质量目录批量提交
- ❌ 链接交换网络
- ❌ 隐藏链接

## 外链监控

### 每月检查

1. 新增外链是否被Google收录
2. 已有外链是否仍然有效（未变为404/nofollow）
3. 是否有垃圾外链指向我们（需要在GSC中拒绝）

### 异常处理

| 异常 | 行动 |
|------|------|
| 发现大量垃圾外链 | 在GSC中提交Disavow文件 |
| 高质量外链消失 | 联系对方确认，尝试恢复 |
| 竞品获得高权威外链 | 分析来源，争取同等机会 |
