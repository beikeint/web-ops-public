# Digital PR · 现代外链建设 v1.0

> **何时用**：客户站权重低（DR < 30）/ 需要 Tier-1 媒体提及 / 转化路径需要 trust signal / 品牌零曝光要破圈
> **不适用**：纯 SEO 内链（用 tech-optimization）/ 内容 SEO（用 content-production）
> **核心原则**：**1 条 Forbes/Reuters 外链 > 100 条目录站垃圾外链**

## 5 大 Digital PR 战术

### 战术 1: HARO / Qwoted / Featured 响应（最高 ROI）

**每日 SOP**（5-10 分钟）：
1. 早 9-10 点收 HARO 邮件（每天 3 封：8am / 1pm / 5:30pm ET = 北京 21:00 / 凌晨 2:00 / 6:30）
2. 扫 query，挑出 **客户专业领域 + 自己有真实数据/案例** 的
3. 在**接到 1 小时内**回复，含：
   - 直接答 query（不绕弯，第一段给核心观点）
   - 具体数据 / 案例 / 引用
   - 可附上专家头衔（如"DesaiInd 总工程师 X 年经验"）
   - 联系方式 + 个人 LinkedIn / Wikipedia
4. 记 outreach log（哪个 query / 回复时间 / 是否被采用）

**模板（B2B 化工示例）**：
```
Hi [记者名],

[一句话答 query 的核心]

[具体数据/案例支撑,带数字]

[引用一个具体客户/项目,带实际数字]

I'm [Name], [Title] at [Company], [N] years in [domain]. We work with [client examples].
Happy to provide further data/quotes for [topic] if needed.

[Email]
[Phone]
[LinkedIn URL]
[Company website]
```

### 战术 2: Unlinked Brand Mentions 抓回（最低成本高回报）

**步骤**：
1. **每月扫一次** Google 搜：`"<品牌名>" -site:<自己网站>`
2. 拿前 50 条结果
3. 筛选：是否提到品牌但没加链接？
4. 找文章作者邮箱（用 Hunter.io / Apollo.io）
5. 发外联邮件：

```
Hi [Author],

I noticed you mentioned [Company] in your article "[Article Title]"
on [Date]. Thanks for the shoutout!

Would you mind adding a link to our site so readers can learn more?
URL: [Most relevant page]

Happy to return the favor or share the article on our LinkedIn (we
have [N]K followers).

Thanks,
[Name]
```

**预期成功率**：30-50%（已经提了品牌的人对加链接阻力小）

### 战术 3: Resource Page Outreach

**步骤**：
1. Google 搜：`"<行业关键词> resources"` / `"<行业> tools"` / `"<行业> guides"` / inurl:resources
2. 找 DR > 30 的资源页
3. 评估你的内容是否值得加入
4. 发邮件：

```
Hi [Editor],

I came across your resource page on [topic] — really useful collection!

Noticed you have [Resource X] which is great. We've recently published
[Your Resource] which covers [unique angle]. It might be a useful
addition for your readers.

Link: [URL]
Why it's worth including: [1-2 sentence pitch]

Either way, thanks for putting together such a great resource!

Best,
[Name]
```

### 战术 4: Broken Link Building

**步骤**：
1. 用 Ahrefs / Semrush 找目标行业站的 broken outbound links
2. 替代品检查：你的内容是否能替代？
3. 发邮件提醒站长：

```
Hi [Webmaster],

While reading [Page Title], I noticed [link X] is broken (404).

Just a heads up. If you're looking for a replacement, we have
[Your Content] which covers similar ground:
[URL]

Either way, thought you'd want to know!

Cheers,
[Name]
```

**预期成功率**：5-15%（要看是否真有替代价值）

### 战术 5: 品牌实体建设（GEO 时代关键）

**目标**：在 ChatGPT / Perplexity 知识图谱里建立"实体存在"

**核心动作**：
1. **Wikipedia 词条**（创始人 / 公司 / 行业贡献）— 难度高但权重最大
2. **Wikidata 实体**（每个 Wikipedia 词条需要对应 Wikidata Q-id）
3. **LinkedIn 公司页 + 创始人页**（完整填，含 about / founder note / employee count / website）
4. **YouTube 频道**（即使只发 1 个工厂介绍视频也要建）
5. **Reddit 社区参与**（不硬广，回答相关问题）
6. **Crunchbase 公司档案**
7. **G2 / Capterra**（如果是 SaaS）/ **行业目录**（如果是 B2B 制造业）

**为什么？**：
- 2025 Ahrefs 研究 75K 品牌：**品牌提及与 AI 引用相关性是反链的 3 倍**
- ChatGPT 47.9% 引用 Wikipedia
- Perplexity 46.7% 引用 Reddit
- 11% 域名同时被 ChatGPT 和 Google AIO 引用 → 平台分别建设

## Outreach 节奏与目标

**单客户每周节奏**：
- HARO/Qwoted 响应：5-10 条 query（早晚各 1 次邮件检查）
- Unlinked mention：每月 1 次扫描，10-20 条外联邮件
- Resource page：每月 5-10 条外联
- Broken link：每月 5-10 条外联
- 品牌实体维护：持续，每月 1 次审计

**KPI 目标**：
- 月新增 backlink：2-5 条 Tier-1 / 5-10 条 Tier-2
- 月新增 brand mention（含/不含链接）：10-30 次
- DR 增长：3-6 个月内 +5-10

## v10 起步 30 天计划

### 第 1 周：基础设施
- 注册 Featured.com 账号（每客户 1 个）
- 设置邮件订阅（北京时间 2:00 / 6:30 检查）
- 写 outreach 邮件模板（每客户行业版）

### 第 2 周：HARO 系统化
- 每天 09:00 跑 HARO query 筛选 SOP
- 1 小时内回复目标 query

### 第 3 周：Unlinked Mention 启动
- 用 Google Alerts 设置每客户品牌名 alert
- 每周扫一次结果，启动外联

### 第 4 周：实体建设深化
- LinkedIn 公司页完整度审计 + 补全
- 创始人 LinkedIn 个人页内容化（每周 2 条 post）
