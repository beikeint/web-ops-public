激活网站运营智能体 v9.4，对客户 **$1** 跑完整的每日巡检流程。

执行 `.claude/skills/daily-ops.md` 全流程：
1. **基础巡检 7 项**：响应/SSL/sitemap/robots/索引/Core Web Vitals/搜索曝光
2. **第 8 项交叉检查**：GSC 展示>0 但线上 404 的页面（每周一必跑）
3. **增长 7 维度**：CTR / 内链 / Schema / 内容更新 / 竞品对比 / AI 引用 / GEO 信号
4. **闭环原则**：发现问题**当天修复 + 验证**，禁止只出报告
5. **MCP 数据交叉验证**：seo-checker MCP 检测结果用 fetch 二次验证（重定向页数据不可靠）
6. **客户简报**：调用 client-manager 时间线记录今日里程碑 + 出客户能看懂的简报

输出格式：
- 巡检表格（每项 ✅/⚠️/❌ + 已修复/已验证状态）
- 增长 7 维度发现（按机会价值排序）
- 已闭环的修复清单
- 给 $1 客户的简报（中文，过 humanizer-zh 去 AI 味）

**自我进化**：本轮发现的本应更早发现的问题，必须按 4 步进化机制更新 daily-ops.md / QA 清单 / 时间线。
