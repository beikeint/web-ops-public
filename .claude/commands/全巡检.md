description: 一键跑所有活跃客户的每日巡检（顺序跑完，统一汇总）
argument-hint: (无参数)
---

激活网站运营智能体 v9.4，对**所有活跃客户站**顺序跑每日巡检。

**执行流程**：
1. 调用 client-manager MCP `list_clients` 拉取所有 status=active 的客户
2. 对每个客户跑 `daily-ops.md` 全流程（基础 7+1 + 增长 7 维度）
3. 发现的问题**当天闭环修复**，禁止积累
4. 出**汇总日报**：每客户一行表格（站名 / 状态 / 今日发现 / 已闭环动作 / 客户简报已发?）

**汇总输出**：

```
| 客户 | 巡检状态 | 关键发现 | 已闭环 | 简报 |
|---|---|---|---|---|
| client-B | ✅ | CTR 机会页 3 个 | Title/Desc 已重写 | ✅已发 |
| demo-b | ⚠️ | sitemap 提交失败 | 重新提交+IndexNow 推 | ✅已发 |
| ... | ... | ... | ... | ... |
```

**严禁**：跳过任何客户、只出报告不修复、对一个客户做一半就跑下一个。

**全跑完**才能收尾，最后给我一份"今日全客户巡检完结报告"。
