激活网站运营智能体 v9.4，给客户 **$1** 出一篇关于「**$2**」的博客。

执行 `.claude/skills/content-production.md` 全流程：

1. **选题验证**：
   - 调 search-analytics MCP 看该关键词 GSC 是否已有展示数据（避免重复内容）
   - 调 seo-sxo skill 反向读 SERP，判断"$2"在 Google 是哪种 page-type 主导（产品页 / 对比 / how-to / 信息文章）
   - 如果 SERP 主导是产品页/对比页，**警告运营人员博客文章可能突破不了**，建议改 page-type 或换关键词

2. **内容生产**（page-type 确认是文章型才继续）：
   - 标题（含主关键词，不夸张，符合 humanizer-zh 第 1/2/8 项规则）
   - 段首 60 字给定义（GEO 优化，被 AI 引用关键）
   - 1-2 个 200-260 字"自包含答案块"（seo-geo skill 要求）
   - H2/H3 用问句式标题
   - 加具体统计数据 + 来源
   - FAQ 区块（3-5 问）
   - 内链 3-5 个相关页面

3. **SEO 元数据**：title ≤ 60 字、desc 150-160 字、canonical、og:image、Schema Article + Person

4. **强制过 humanizer-zh** 去 AI 味（24 种模式检测）

5. **图片**：
   - 配 1 张主图（按"图片红黄绿三线规则"决定红线/黄线/绿线 → AI 生成 / 合成 / HTML）
   - 调 image-generator MCP `generate_cover` 出主图
   - alt 含主关键词

6. **发布前 QA**：调 seo-checker MCP `check_seo` 自检；如客户站多语言，调 `check_hreflang` 跑 8 项

7. **time line**：client-manager 记录"今日发布博客《$2》"

**输出**：完整博客 markdown + 主图 + 元数据 + 客户预览说明
