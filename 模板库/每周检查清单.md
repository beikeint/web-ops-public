# 每周检查清单 — 轻量级运营监控

> 每周3次，每次5分钟，确保网站持续健康运行  
> 适用：所有客户，从上线第一周开始执行

---

## 周一：流量与索引检查

**目标**：确认上周流量正常，无异常波动

### 检查步骤

```
周一检查

请对当前客户执行周一检查：

1. GA4上周流量概览：
   - 上周UV/PV（如有权限或CSV）
   - 与前一周对比，是否有异常波动（>30%变化）
   - 自然搜索流量是否正常

2. Google索引状态：
   - 搜索 site:{{域名}}，记录索引页面数
   - 与上次检查对比，是否有页面掉出索引

3. 网站可访问性：
   - 访问首页，确认加载正常
   - 访问最新博客文章，确认可访问
   - 检查是否有SSL证书警告

如发现异常，立即报告。无异常则记录"周一检查通过"。
```

### 检查清单

- [ ] GA4流量无异常波动
- [ ] 索引页面数未减少
- [ ] 网站首页加载正常
- [ ] 最新博客可访问
- [ ] SSL无警告

**耗时**：约5分钟  
**输出**：一行状态记录到 `客户-XX/优化/[YYYY-MM]_周检查记录.md`

---

## 周三：询盘与转化检查

**目标**：追踪本周询盘情况，确认表单正常

### 检查步骤

```
周三检查

请对当前客户执行周三检查：

1. 表单功能验证：
   - 访问联系页面，确认表单显示正常
   - 检查Web3Forms服务状态（如可访问）

2. 询盘数据（如有）：
   - 本周收到的询盘数量
   - 询盘来源页面（哪个页面的表单）
   - 买家国家分布

3. GA4转化事件：
   - form_submit事件本周触发次数
   - 转化率是否正常

如表单异常，立即通知。
```

### 检查清单

- [ ] 联系页表单显示正常
- [ ] 本周询盘数量已记录
- [ ] 转化事件正常触发
- [ ] 无异常的0询盘情况（排除表单故障）

**耗时**：约5分钟

---

## 周五：竞品与内容检查

**目标**：快速扫描竞品动态，检查内容表现

### 检查步骤

```
周五检查

请对当前客户执行周五检查：

1. 竞品快速扫描：
   - 访问客户配置中的TOP3竞品博客页面
   - 是否有新发布的文章（标题+日期）
   - 如有重要更新，标记为下月分析重点

2. 我方内容表现：
   - 最新发布的博客文章是否已被Google索引（site:域名/文章URL）
   - Search Console中是否出现新的搜索查询

3. GEO快速检查：
   - 搜索我方核心关键词，检查AI概览（SGE）中是否引用了我方内容
   - 如未被引用，标记需要优化的文章

无重要变化记录"周五检查通过"。有变化则详细记录。
```

### 检查清单

- [ ] 竞品无重大内容更新（或已标记）
- [ ] 最新博客已被Google索引
- [ ] AI搜索结果检查完毕
- [ ] 本周检查记录已归档

**耗时**：约5分钟

## GEO每月必做动作

每月月初（与指令1同步执行）额外完成以下GEO优化：

### 1. llms.txt检查与更新

```
GEO检查

请检查 {{域名}}/llms.txt 是否存在：
- 如存在：检查内容是否包含最新的产品和博客信息
- 如不存在：建议创建，包含公司简介、核心产品、最新文章列表

llms.txt内容格式：
# [公司名]
> [一句话描述]

## Products
- [产品1]: [描述]
- [产品2]: [描述]

## Recent Articles
- [文章1标题]: [URL]
- [文章2标题]: [URL]
```

### 2. FAQ内容扩充

每月至少为2个页面补充新的FAQ问题：
- 来源：Search Console中的疑问句式搜索词
- 来源：Google "People Also Ask" 中的新问题
- 来源：客户收到的实际询盘中的高频问题

### 3. 结构化数据审计

```
结构化数据检查

请检查以下Schema标记的完整性：
1. Organization Schema：公司信息是否最新
2. Product Schema：所有产品页是否有Product标记
3. Article Schema：所有博客是否有Article+FAQ标记
4. BreadcrumbList：面包屑导航Schema是否正确
5. FAQ Schema：FAQ部分是否有结构化标记

使用Google Rich Results Test验证。
```
