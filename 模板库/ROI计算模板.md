> 帮客户算清这笔账：运营投入值不值

## 一、外贸独立站ROI计算

### 1.1 计算公式

```
ROI = (产出价值 - 运营投入) / 运营投入 × 100%

产出价值 = 询盘数量 × 有效率 × 成交率 × 平均订单金额
运营投入 = 月服务费 × 月数
```

### 1.2 标准案例（标准版客户）

| 项目 | 数值 | 说明 |
|------|------|------|
| **投入** | | |
| 月运营费 | ¥4,980 | 标准版 |
| 季度投入 | ¥14,940 | 3个月 |
| **产出** | | |
| 季度总询盘 | 15条 | 月均5条（运营6个月后预期） |
| 有效询盘率 | 60% | 9条有效询盘 |
| 成交率 | 15% | 约1.35单≈1单 |
| 平均订单金额 | ¥50,000 | 外贸B2B常见客单 |
| **季度成交金额** | **¥50,000** | |
| **ROI** | **¥50,000 / ¥14,940 = 334%** | **每投入1元获得3.34元回报** |

### 1.3 保守/标准/乐观场景

| 场景 | 月询盘 | 季度成交 | 成交金额 | 季度ROI |
|------|--------|----------|----------|---------|
| 保守 | 2条 | 0.5单 | ¥25,000 | 67%（亏损期） |
| 标准 | 5条 | 1单 | ¥50,000 | 234% |
| 乐观 | 10条 | 2单 | ¥100,000 | 569% |

> **关键洞察**：即使在保守场景下，只要半年内成交1单（¥50,000），年化ROI也远超100%。外贸B2B的特点是单价高、频次低——一单就能覆盖全年运营费用。

| 获客渠道 | 单询盘成本 | 询盘质量 | 持续性 | 可控性 |
|----------|-----------|----------|--------|--------|
| **网站运营智能体运营** | ¥{{X}} | 高（主动搜索） | 持续（SEO复利） | 高 |
