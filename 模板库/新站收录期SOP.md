# 新站收录期SOP — 前三个月特殊流程

> 适用：所有新上线网站的前3个月  
> 核心理念：新站前3个月是建设期，不是收获期。管理客户预期比追求数据更重要。

---

## 总览：三个月阶段目标

```
第1个月              第2个月              第3个月
收录建立期            内容积累期            排名突破期
┌──────────┐      ┌──────────┐      ┌──────────┐
│ 目标：     │      │ 目标：     │      │ 目标：     │
│ Google收录 │  →   │ 关键词覆盖 │  →   │ 核心词前50 │
│ 所有页面   │      │ 面扩大     │      │            │
│           │      │           │      │            │
│ 预期流量： │      │ 预期流量： │      │ 预期流量：  │
│ 极少(正常) │      │ 开始出现   │      │ 稳定增长   │
│           │      │           │      │            │

└──────────┘      └──────────┘      └──────────┘
```

## 第1个月：收录建立期

### 核心目标

让Google索引网站所有页面。没有被索引的页面不可能获得排名和流量。

### 每周必做

| 周次 | 检查项 | 方法 | 预期结果 |
|------|--------|------|----------|
| Week 1 | 提交Sitemap | GSC → 站点地图 → 提交 | 状态显示"成功" |
| Week 1 | 手动请求索引 | GSC → URL检查 → 对10个重要页面请求索引 | 已提交 |
| Week 2 | 检查收录数 | 搜索 site:域名 | 预期：30-50%页面已收录 |
| Week 3 | 检查收录数 | 同上 | 预期：50-80%已收录 |
| Week 4 | 检查收录数 | 同上 | 预期：80-100%已收录 |

### 每月必做

1. **发布2-4篇博客**（针对核心关键词的长尾变体）
2. **完成竞品调研**（建立竞品基准档案）
3. **完成关键词研究**（建立初始关键词矩阵）
4. **技术SEO全面检查**（PageSpeed、Schema、hreflang等）

### 月报重点（首月特殊月报）

**展示**：
- 网站健康评分（技术SEO得分高=专业能力证明）
- 已发布内容列表（让客户看到工作量）
- 收录进度（XX页已被Google收录）
- 竞品对标（我们的起点 vs 竞品现状）
- 3个月目标预测

**不展示**：
- 流量数据（太少，会打击信心）
- 关键词排名（还没有，是正常的）
- 询盘数据（不现实）

### 客户沟通话术

**客户问："网站上线一个月了，怎么没有流量？"**

> "XX总，这是完全正常的。新网站就像新开的店铺——Google需要时间来'发现'和'信任'您的网站。好消息是，我们已经完成了所有技术配置，Google已经开始收录我们的页面（目前已收录XX页/总共XX页）。同时我们已经发布了X篇行业博客，这些内容通常需要2-3个月才能开始在搜索结果中出现。根据同行业竞品的数据，预计第3个月您会开始看到自然流量，第6个月会有稳定的询盘。我们正在按计划推进。"

**客户问："竞品已经排在前面了，我们能追上吗？"**

> "可以。竞品虽然先发优势明显，但我们在分析中发现了他们的明显弱点：[具体弱点，如：仅英文版本/无深度内容/网站技术过时]。我们的策略是不跟他们在数量上竞争，而是用更深度、更多语言、更精准的内容来切入他们没有覆盖的搜索词。这是一个6-12个月的过程，但复利效应会越来越明显。"

## 第2个月：内容积累期

### 核心目标

扩大关键词覆盖面，让更多搜索词开始出现在Search Console中。

### 每周必做

| 周次 | 检查项 | 重点 |
|------|--------|------|
| Week 1 | GSC数据首次分析 | 哪些词开始有展示？（即使没有点击也是好信号） |
| Week 2 | 内容发布+索引 | 确认新博客已被索引 |
| Week 3 | 排名萌芽监控 | 关注50-100名之间的关键词（开始有排名了） |
| Week 4 | 月度数据汇总 | 记录本月所有新出现的搜索词 |

### 每月必做

1. **发布2篇博客**（标准版），覆盖不同长尾词
2. **分析GSC初始数据**：
   - 新出现的搜索词有多少个
   - 哪些词有展示但没有点击（需优化Title/Description）
   - 哪些页面开始获得展示
3. **持续竞品监控**
4. **优化现有页面**（基于GSC数据微调Title/Description）

### 月报重点

**展示**：
- 收录完成情况（理想状态：100%已收录）
- **新出现的搜索词数量**（关键正面信号）
- GSC展示次数（即使点击少，展示多=Google开始认可我们）
- 累计发布内容（博客总数，展示持续投入）
- 内容质量指标（字数、多语言、Schema等）

**开始谨慎展示**：
- 流量数据（标注"建设期数据，仅供参考"）
- 初始排名数据（标注"排名尚不稳定"）

### 客户沟通话术

**客户问："第二个月了，流量还是很少。"**

> "XX总，第二个月的数据我们看到了几个非常积极的信号：第一，Google已经完全收录了我们的XX个页面；第二，Search Console中已经出现了XX个搜索词在展示我们的网站——这意味着Google已经开始把我们的网站推荐给搜索用户了。虽然点击还不多，但展示量在增长。根据SEO的规律，从'开始展示'到'获得点击'通常需要1-2个月。第3个月我们重点优化这些有展示的词，帮助它们进入前10名。"

## 第3个月：排名突破期

### 核心目标

核心关键词进入前50名，长尾词开始进入前10名。

### 每周必做

| 周次 | 检查项 | 重点 |
|------|--------|------|
| Week 1 | 排名机会词识别 | 找出排名11-30位的词（最容易突破） |
| Week 2 | 针对性优化 | 为机会词添加内链、优化页面内容 |
| Week 3 | 效果验证 | 检查优化后排名是否提升 |
| Week 4 | 月度数据分析 | 完整的排名变化趋势 |

### 每月必做

1. **发布2篇博客**（选题基于GSC数据中的机会词）
2. **重点优化排名11-30位的关键词**：
   - 为这些词的对应页面增加内链
   - 优化页面Title和H1（更精准匹配搜索意图）
   - 补充内容深度（如果页面内容太薄）
3. **内链大优化**：将3个月的博客形成内链网络
4. **转化优化**：优化CTA位置和文案（为即将到来的流量做准备）

### 月报重点

**全面展示**：
- 流量趋势图（3个月趋势线，应该是上升的）
- **关键词排名变化**（核心数据，展示进步）
- 前10名/前30名/前50名的关键词数量
- 点击量增长
- 询盘数据（如有）
- 投入产出分析（初步ROI雏形）

### 客户沟通话术

**客户问："什么时候能真正看到效果？"**

> "XX总，效果已经开始显现了。这个月我们有XX个关键词进入了Google前50名，其中XX个已经进入前20名。特别是'[具体关键词]'，已经排到了第XX位。按照目前的趋势，下个月应该能有几个词进入前10名，也就是Google首页。一旦进入首页，点击量会有质的飞跃——前10名获取了90%的搜索点击。同时我们已经发布了XX篇博客文章，这些都是长期资产，会持续为您带来流量。"
